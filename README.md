📌 Team Task Management System

A modern full-stack Task & Team Management Web Application built to help teams collaborate, assign tasks, track progress, and improve productivity.

✨ Features
👥 Team Management
Create and manage teams
Add/remove team members
Role-based access (Owner / Member / Admin support ready)
📋 Task Management
Create, update, delete tasks
Assign tasks to team members
Set priority (Low, Medium, High, Urgent)
Track status (Pending, In Progress, Completed)
Add due dates
📊 Task Insights
Task status tracking
Overdue detection
Activity logs (task history)
🔐 Authentication
User registration & login
Secure session handling (cookie-based auth)
Forgot / reset password support (API ready)
🧱 Tech Stack
Frontend
React.js (Vite)
Tailwind CSS
date-fns
react-hot-toast
Axios
Backend
Node.js
Express.js
Prisma ORM
PostgreSQL / MySQL (configurable)
Authentication
Session / JWT-based auth (depending on setup)
🔐 Role System
Role	Permissions
User	Create & manage own tasks
Admin	Full access (all teams/tasks)
Owner	Manage team + members
🎯 Key Highlights
Clean UI with modern design system
Fully responsive layout
Real-time task updates
Scalable backend architecture
Component-based reusable frontend
🛠 Future Improvements
